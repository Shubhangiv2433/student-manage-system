# Student Management System

A modern and responsive Student Management System built using React.js and Tailwind CSS.
This project helps manage students, authentication, dashboard statistics, and student records with a clean UI.

---

## 🚀 Features

* Admin Login Authentication
* Dashboard UI
* Add Students
* Edit Student Details
* Delete Students
* Search Students
* Responsive Design
* Toast Notifications
* React Router Navigation
* Context API State Management
* Modern Tailwind CSS UI

---

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* Context API
* JavaScript
